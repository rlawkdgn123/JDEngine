솔루션 설정법

pch.h : 솔루션 속성 - 미리 컴파일된 헤더

main을 제외한 라이브러리 설정법

구성 속성 - 일반 - 구성 형식 - 정적 라이브러리(.lib)

종속성　설정　뒤에　디렉터리를　포함시켜주어야　한다．
1. VC++디렉터리 - 포함 디렉터리 - 프로젝트 경로 선택

2. 라이브러리 디렉터리 $(OutDir) $(VC_LibraryPath_x64) $(WindowsSDK_LibraryPath_x64) 추가

3. 링커 - 입력 추가 종속성에 라이브러리명 추가

+추가로

$(VC_IncludePath)
VC 기본 경로 추가도 해주자

$(VC_IncludePath)는 Visual Studio 빌드 시스템에서 미리 정의된 환경 변수(매크로) 중 하나로,

Visual C++ 기본 포함 경로 목록을 나타냅니다.

컴파일러가 기본적으로 참조하는 헤더 파일 경로들이 여러 개 포함되어 있습니다.

예를 들어, Windows SDK, CRT(Standard C++ 라이브러리), ATL/MFC 헤더 경로 등이 들어있습니다.
